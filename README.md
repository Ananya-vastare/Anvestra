# Anvestra – AI-Powered UX Audit Platform

Anvestra is a full-stack AI-powered UX audit platform that helps designers and developers evaluate user interfaces more efficiently. By leveraging the Google Gemini API, the platform analyzes UI screenshots and user feedback to generate structured usability insights and actionable design recommendations. Built with React.js, Node.js, and Express.js, Anvestra provides a responsive dashboard for reviewing evaluation results and streamlining the UX improvement process.

##  Features

- AI-powered UX audits using the Google Gemini API
- Analyze UI screenshots and user feedback
- Generate structured usability recommendations automatically
- Interactive dashboard for visualizing UX insights
- Asynchronous backend processing for efficient request handling
- RESTful API integration between frontend and backend
- Responsive, modern interface built with React and Tailwind CSS
- Actionable design recommendations to improve user experience
- Modular and scalable full-stack architecture
